================================================================================
                    SX1280 测距工程 (HC32L021)
================================================================================

本目录为 Keil 工程壳，固件源码在 ../../sx1280/ 。

快速开始
--------------------------------------------------------------------------------
1. 打开 MDK/sx1280_ranging.uvprojx
2. 修改 sx1280/app/main.c 中 DEMO_SETTING_ENTITY（MASTER / SLAVE）
3. 编译、烧录

Keil 工程分组
--------------------------------------------------------------------------------
  common          启动、system、中断、板级
  hc32-ddl        HC32 外设库 (SPI/GPIO/...)
  sx1280-driver   Semtech SX1280 驱动
  sx1280-hal      HC32 平台移植 (SPI/IRQ/按键)
  sx1280-ranging  测距状态机 + 门控 + 发布
  sx1280-display  ST7735S + 测距 UI
  sx1280-main     main.c

详细文档
--------------------------------------------------------------------------------
  仓库根目录 README.md
  sx1280/README.md
  sx1280/docs/RANGING_PHASE_A.md

注意事项
--------------------------------------------------------------------------------
  source/ddl_device.h 为 HC32 DDL 必需头文件，勿删。
  所有应用源码仅维护 sx1280/ 目录，勿在 project/ 下复制源码。
