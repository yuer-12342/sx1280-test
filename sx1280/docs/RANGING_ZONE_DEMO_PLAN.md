# 稳定判区 DEMO 方案（放弃绝对精度）

> **版本：** v1.0 · 2026-07  
> **适用：** XL1280-SMT 双板（MASTER+TFT / SLAVE），SF7+BW800，cal=11550  
> **目标：** 固定真距读数带稳定 + 远近读数带可分开 + 驱动报警  
> **不验收：** 屏显米数 = 卷尺；正式标定冻结；15～30 m 绝对精度  

---

## 1. 验收口径（对外一句话）

> 验证 LoRa 2.4G 在项圈近地场景下，**固定距离读数不乱漂**，**远近可分档并触发围栏报警**；  
> 显示米数为**工程相对量**，允许整体偏差，**不承诺** GPS/卷尺级精度。

---

## 2. 核心原则

| 原则 | 说明 |
|------|------|
| **只看判区，不看精度** | 3 m 真距显示 2 m、6 m 显示 5 m 均可接受 |
| **统计用 Rnd，不用 Dist** | Hold 时 Dist 卡住旧值；Rnd 每轮更新，反映当前空口 |
| **每档独立上电** | 避免 Jump 门控；拉远动态演示除外 |
| **顺序 S → Z → A** | 近区不稳不做 Z；判区不过不做 A |
| **Gate 记录完整** | Hold 行记 Spread/Jump/dRnd，便于复盘 |

## 3. 测试几何与记录

### 3.1 标准摆法（主验收）

```
侧视：

  MASTER                         SLAVE
  凳高 H（见下表）                同高 H
  竖直，ANT 记号面对面
  ═══════ 凳 ═══════════════════ 凳
  ───────────── 地面 ─────────────
```

| 项 | 规定 |
|----|------|
| 凳高 H | **当前摸底：1.0 m**（MASTER/SLAVE 同高，差 < 3 cm）；正式近地验收可另补 **0.45 m** 一组 |
| 姿态 | 竖直、板面平行、ANT/3# 记号面对面 |
| 真距 | 卷尺 ANT 记号 → ANT 记号，与地面平行 |
| 场地 | 开阔；离墙/栏杆 ≥ 真距；天台瓷砖需注明 |
| 人体 | 测距时退到链路侧面 ≥ 3 m |
| 录屏 | 手机架 MASTER 屏侧，不在两板连线上 |

### 3.2 上电与预热

```
1. 两板断电 ≥ 30 s
2. 先 SLAVE，等 10 s
3. 再 MASTER
4. 弃前 8 轮；正式记录从第 9 轮起 No=1
```

### 3.3 单点记录行

```
No | 时间 | Stat | Dist | Rnd | Hop | Gate | RSSI | 备注
```

**统计规则：**

- **S / Z 正式统计：** 优先 **Stat=Done 且 Gate=OK** 的 `Rnd`
- **Done% 不足时（Z 备选）：** 可用 `Rnd > 0` 且 `Gate ≠ INV` 的有效行，表头注明
- **Hold 行：** 单独计数 Spread / Jump / dRnd，不计入稳定极差

### 3.4 表头（每组必填）

```
日期：________  时段：________  场地：________（□瓷砖 □水泥 □草地）
天气：________  固件 Git：________  cal：11550
MASTER SN：________  SLAVE SN：________
凳高 MASTER / SLAVE：_____ m / _____ m
泡沫垫：□无  □有

```

## 5. S 项 — 近区稳定（弱化版，不看绝对精度）

**目的：** 3 m 真距时读数带不乱漂，防止近区误报。

**条件：** 真距 **3.0 m**，摆好后人退开，板不动。

### 5.1 S-快测（15 min，可先跑）

| 步骤 | 操作 |
|------|------|
| 1 | 独立上电，摆 3 m |
| 2 | 弃 8 轮 |
| 3 | **1 min/点，共 15 点**，抄 Rnd/Gate |
| 4 | 只统计 Done+OK 的 Rnd |

| ID | 指标 | 通过线 |
|----|------|--------|
| Sq-a | Rnd 极差 | ≤ **1.5 m** |
| Sq-b | Rnd < **1.0 m** 点数 | **0** |
| Sq-c | Done+OK 占比 | ≥ **50%** |

**Sq 过 → 可做 Z；Sq 不过 → 加泡沫垫/离墙更远后重测。**

### 5.2 S-正式（可选，签字用）

在 Sq 基础上补测至 **30 min / 30 点**（或次日再 15 点做重复性）：

| ID | 指标 | 通过线 |
|----|------|--------|
| S-a | Rnd 极差（Done+OK） | ≤ **1.5 m** |
| S-b | 80% 点落在 `[均值±0.8]` 内 | 是 |
| S-c | Rnd < 1.0 m 点数 | **0** |
| S-d | 连续 5 点单调下降且总降 > 1.0 m | **不得出现** |
| S-e | 两日/两段均值差 | < **0.8 m**（可选） |

**S 总评：** S-a～S-d 全过（S-e 可选）。

## 6. Z 项 — 稳定判区（核心，不看精度）

**前提：** S 快测 Sq 或 S 正式已过。

**目的：** 近/远读数带分开，精度只需「合理」（远带明显高于近带）。

### 6.1 采点步骤（每档必须独立上电）

| 步骤 | 卷尺真距 | 操作 | 输出 |
|------|----------|------|------|
| **Z-3** | **3 m** | 断电上电 → 弃 8 轮 → 15 点（10 s/点） | **M_near** = Done+OK 的 Rnd 中位数 |
| **Z-7** | **6～8 m**（卷尺量准，建议 **7 m**） | **重新断电上电** → 同上 | **M_mid** |
| **Z-10** | **10 m**（场地不够可跳过） | **重新断电上电** → 同上 | **M_far** |

> **禁止：** 从 3 m 拉到 7 m 后连续记点（会触发 Jump，Dist 卡住）。

每档记录：

```
真距：___ m
M___ = ___ m
极差：___ m
Done+OK：___ / 15（___%）
最低 Rnd：___   最高 Rnd：___
Spread 轮数：___  Jump 轮数：___
```

### 6.2 通过线

| ID | 指标 | 通过线 |
|----|------|--------|
| **Z1** | 判区分离 | **M_mid ≥ M_near + 1.5 m** |
| **Z2** | 远区分离（有 Z-10 时） | **M_far ≥ M_mid + 1.0 m** |
| **Z3** | 每档 Done+OK 比例 | ≥ **50%**（理想 ≥ 60%） |
| **Z4** | 远档最低 Rnd | ≥ **M_near + 0.8 m** |

**读数带示意：**

```
Rnd
  |        ┌── 7 m 带 ──┐
  |   ┌─ 3 m 带 ─┐      │
  +---+----------+--------+--→ 真距
      3 m        7 m
      ←─ 间隙 ≥ 1.5 m（读数单位）─→
```

**Z 总评：** Z1 + Z3 + Z4 必过；有 Z-10 时加 Z2。

**Z 不过：** 结论「当前几何/场地下无法稳定判区」，不演示报警。

## 7. 报警阈值（由 Z 数据算出）

Z 通过后，填入：

```
M_near = ______ m
M_mid  = ______ m

T_clear = M_near + 0.8 m  = ______ m   （低于 → 倾向 Safe）
T_alarm = M_mid  - 0.5 m  = ______ m   （高于 → 倾向 OUT!）
```

**固件：**

- 统计/验收用 **Rnd**
- 报警建议 `FENCE_ALARM_INDOOR_DEMO = 1`（用 Rnd，Hold 也计数）  
  或按 T_alarm/T_clear 配置（F2，Z 通过后实现）

**屏 Set 与真距映射（DEMO 操作，非卷尺）：**

| 屏 Set | 实际操作真距 | 用途 |
|--------|--------------|------|
| 3 | ~3 m | 近区 / 解除 |
| 5 | ~7 m | **主测拉远** |
| 10 / 15 | ~10 m | 可选 |

## 8. A 项 — 报警 DEMO

**前提：** Z 已过，T_alarm / T_clear 已填写。

### 8.1 测试步骤

| ID | 操作 | 预期 | 通过线 |
|----|------|------|--------|
| **A1** | 真距 ~3 m，停 **1 min** | Alm = **Safe**，无 OUT! | 全程 Safe |
| **A2** | 从 3 m **匀速拉到 ~7 m**，停稳 | OUT! + 蜂鸣 | ≤ **15 s**（理想 ≤ 10 s） |
| **A3** | 从 7 m **回到 ~3 m**，停稳 | Safe | ≤ **15 s** |
| **A4** | 长按 ≥ 3 s（Mute） | 较长嘀一声，蜂鸣停/报警开关 | 功能正常 |
| **A5** | 短按 Set | 3→5→10→15 循环，短嘀 | 功能正常 |

**A2、A3 各做 2 轮**，均达标。

### 8.2 A 项记录

```
T_clear = ___   T_alarm = ___
判距来源：□ Rnd  □ Dist

A2 第1轮：延迟 ___ s   A2 第2轮：___ s
A3 第1轮：延迟 ___ s   A3 第2轮：___ s
```

**说明：** 动态拉远时前几轮可能 Hold；用 Rnd 判距时，Rnd 到 5～7 m 即应触发逻辑。

## 12. 常见问题与对策

| 现象 | 含义 | 动作 |
|------|------|------|
| 3 m Rnd 1～3 m，极差 ~2 m | 精度偏差大但可能有带 | Sq 放宽到 1.5 m；看 Z1 是否仍过 |
| 7 m Hold，Rnd 5～7 m | 空口有远距信息，门控挡 Dist | **Z 用 Rnd**；A 用 Rnd |
| Gate = Spread | 远距 hop 散 | 停稳 3 min；泡沫垫；必要时 Spread 0.5→0.65 |
| Gate = Jump | 带着近距历史测远距 | **每档独立上电** |
| Z1 过但 Done% < 50% | 判区有信号但不稳 | 改善场地；或记「有条件通过」 |
| Z 不过 | 远近带重叠 | Track C：只演示 UI，不演示报警 |
