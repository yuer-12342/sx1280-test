# 7 月 DEMO 方案：近地项圈 · 稳定带 + 判区 + 报警

> **版本：** v2.1 · 2025-06  
> **适用：** XL1280-SMT 双板（MASTER+TFT / SLAVE 项圈），SF7+BW800，cal=11550  
> **对齐 CP罗 口径：** 测距精度可不高；**漂移不能过于严重**；LoRa 在 **1～5 m 本就不稳**（产品不押 1 m 档）；  
> **核心验收：** 固定真距时读数在 **合理误差带内稳定** + **远近读数带可分开（可信判区）** + 驱动报警。

---

## 1. CP罗 要求 → 可测指标

| CP罗 表述 | 工程翻译 | 测什么 |
|-----------|----------|--------|
| 精度可以不高 | 允许整体偏差（如真 3 m 常显 2.5 m） | **不要求** Dist = 卷尺 |
| 不要漂移过于严重 | 真距不变时读数应在一带内，勿越测越偏 | **S 项：稳定性** |
| LoRa 1～5 m 无法稳定 | 近距放宽；安全区按 ≥3 m 真距设计 | 不验 1 m 档 |
| 合理误差内稳定 + 可信判区 | 各真距有稳定读数带，远带高于近带 | **Z 项：判区** |
| 能报警 | 走远进入远带 → OUT! | **A 项：报警** |

**不验收：** 屏上米数 = 卷尺；15～30 m 绝对精度；阶段 A 正式标定签字；16 档 `D_work` 冻结。

## 4. 测试几何（双板同高 0.45 m）

**当前条件：** MASTER 与 SLAVE **均为凳高 0.45 m**（同型塑料凳），与项圈近地高度一致。  
7 月 **全部 S/Z/A 验收均在此摆法下完成**，不依赖墙挂主机。

### 4.1 标准摆法（主验收，必做）

```
侧视：

  MASTER 0.45 m                 SLAVE 0.45 m
       ┌──┐                           ┌──┐
       │  │ 竖直，ANT 记号朝 SLAVE      │  │ 竖直，ANT 记号朝 MASTER
       └──┘                           └──┘
  ════════ 塑料凳 ════════════════════════ 塑料凳
  ─────────────── 地面 ─────────────────────
```

| 项 | 规定 |
|----|------|
| 凳高 | MASTER、SLAVE **均为 0.45 m**（同凳同高，差 < 3 cm） |
| 姿态 | 两板 **竖直**，板面 **平行**，ANT/3# 记号 **面对面** |
| 真距 | 卷尺 **ANT 记号 → ANT 记号**，与地平行 |
| 场地 | 开阔；优先草地/软地；离墙 ≥3 m |
| 人体 | 退到链路侧面 **≥3 m**；录屏手机架 MASTER **屏侧**，不在连线上 |
| 角色 | 带 TFT = MASTER；无屏 = SLAVE（项圈） |

**产品含义：** 主机与项圈均在 **近地高度** 验证；CP罗 验收的是 **「近地场景下能否稳定 + 判区」**，非理想墙挂几何。

### 4.2 可选补测 **G1（主机墙挂，非 7 月必做）**

若后续有条件：MASTER 墙挂 ~1 m + SLAVE 凳 0.45 m。  
**7 月签字不依赖 G1**；仅作产品形态补充数据。

### 4.3 附录 **G_ref（加高对称对照，非验收）**

凳+箱加高至 ~1 m 双板对称：研发用，判断芯片在较好 RF 下是否工作。  
**不得**用 G_ref 通过代替 §4.1 标准摆法。

## 6. 验收项：S → Z → A（顺序不可跳）

```
  S 稳定性（3 m 固定）
       │ 不过 → 调摆位/门控，不测 Z/A
       ▼
  Z 判区（3 m / 8 m / 12 m 读数带分离）
       │ 不过 → 不演示报警；汇报 CP罗
       ▼
  A 报警 DEMO + U UI
```

---

### 6.1 S 项 — 稳定性（抗漂移）

**S1：标准摆法（双板 0.45 m），真距 3 m，30 min**

- 1 min / 点，共 30 点（或录屏回抄 30 个时刻）
- 人退开，板不动

| ID | 指标 | 通过线 |
|----|------|--------|
| S1a | 正式段 `Rnd` 极差（Done+OK） | ≤ **1.0 m** |
| S1b | 80% 点落在 `[均值−0.6, 均值+0.6]` 内 | 是 |
| S1c | `Rnd` < **1.5 m** 的点数 | **0** |
| S1d | 连续 5 点单调下降且总降 > **0.8 m** | **否**（出现则不过） |

**S2：同标准摆法，再测 15 min（或第二日 15 点）**

| ID | 指标 | 通过线 |
|----|------|--------|
| S2 | 两次 S1 的 `Rnd` 均值差 | < **0.5 m** |

**S 总评：** S1a～S1d + S2 全过。

---

### 6.2 Z 项 — 可信判区（读数带分离）

**前提：** S 已过。标准摆法，各 **15 点**（弃 8 轮，5～10 s/点）。

| 步骤 | 卷尺真距 | 输出 |
|------|----------|------|
| Z-3 | **3 m** | 读数带：中位数 **M_near**（Done+OK 的 Rnd） |
| Z-8 | **8 m** | **M_mid** |
| Z-12 | **12 m**（现场最远 10 m 则填 10 m） | **M_far** |

| ID | 指标 | 通过线 |
|----|------|--------|
| Z1 | 判区分离 | **M_mid ≥ M_near + 1.5 m** |
| Z2 | 远区分离（有 Z-12 时） | **M_far ≥ M_mid + 1.0 m** |
| Z3 | 各点 Done+OK 比例 | 每档 ≥ **60%** |
| Z4 | 8 m 档最低 Rnd | ≥ **M_near + 0.8 m**（远带不掉进近带） |

**读数带示意（不要求绝对米数）：**

```
Rnd
  |      ┌─ 8 m 带 ─┐
  |  ┌─ 3 m 带 ─┐   │
  +--+-----------+---+--→ 真距
     3 m         8 m
     ← 两带之间间隙 ≥ 1.5 m（读数单位）→
```

**Z 总评：** Z1 + Z3 + Z4 必过；有 12 m 时加 Z2。

**Z 不过：** 6 m/8 m 读数仍 ~1.x、与 3 m 带重叠 → 对 CP罗 结论 **「近地无法可信判区」**。

---

### 6.3 A 项 — 报警 DEMO

**前提：** Z 已过。用 Z 测得的档心建阈值（**非卷尺 Set**）：

```
T_clear = M_near + 0.8 m    （低于此 → 倾向 Safe）
T_alarm = M_mid  - 0.5 m    （高于此连续 3 轮 Done → OUT!）
```

| 屏 Set | DEMO 操作真距 | 说明 |
|--------|---------------|------|
| 3 | ~3 m | 近区 / 解除 |
| 5 | ~8 m | **主测拉远** |
| 10 / 15 | ~10～12 m | 可选加测 |

| ID | 操作 | 预期 | 通过线 |
|----|------|------|--------|
| A1 | 标准摆法，真距 ~3 m，1 min | Safe | 无误 OUT! |
| A2 | 拉到 ~8 m 停稳 | OUT! + 蜂鸣 | ≤ **10 s** |
| A3 | 回 ~3 m | Safe | ≤ **10 s** |
| A4 | 长按 Mute | 蜂鸣停 | 功能正常 |
| A5 | 短按 Set | 3→5→10→15 | 每档滴一声 |

A2、A3 各 **2 轮**，均达标。

**固件说明（F2，Z 通过后实现）：** 报警优先用 **Rnd** 与 **T_alarm / T_clear**（或基线 + Δ），Set 预设可映射为 Δ，详见 §8。
